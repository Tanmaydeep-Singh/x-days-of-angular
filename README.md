# " x-days-of-angular" 

## Creating Angular App:
- Globally install Angular:
`` npm install -g @angular/cli ``

- Create Angular app:
`` ng new MYAPPNAME ``
    
- Serve app:
`` ng serve ``
    
- Create component:
`` ng generate component MYCOMPONENT ``

- Create service:
`` ng generate service service-name ``

- Create module:
`` ng generate module module-name ``
